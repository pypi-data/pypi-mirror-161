Installation (Gnu/Linux)
========================

You can install M2M Daemons with these commands::

    conda deactivate
    python3 -m venv env
    source env/bin/activate
    python3 -m pip install --upgrade pip
    python3 -m pip install -r requirements.txt
    pip install m2m
